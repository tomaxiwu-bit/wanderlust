## 录制检查

- 录屏分辨率至少 1440×900，隐藏浏览器书签栏和个人信息。
- 日程页至少展示 3 个地点，地图页路线必须可见。
- 预算页至少添加 3 笔不同分类的支出，确保饼图和柱状图有内容。
- 录制后，在 README 的「演示视频」段落填写视频地址、录制日期和应用版本。
- 如果 Supabase 未配置，不要演示登录/同步功能，旁白说明"本地优先，可选云同步"。
- 地图瓦片如果加载慢，可以提前预热缓存，或旁白说明"使用免费 OpenStreetMap 瓦片"。

## 推荐工具

- **OBS Studio**（免费开源）或 **Windows 自带录屏**（Win+G）
- 录制格式 MP4，1080p 或 720p
- 可上传到 B 站或 YouTube，把链接填入 README
